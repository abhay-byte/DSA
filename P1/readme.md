# Two Sum Problem

## Description

The Two Sum problem involves finding two indices in an array of integers where the numbers at those indices add up to a specified target value. Each input is guaranteed to have exactly one solution, and the same element cannot be used twice.

## Problem Statement

Given an array of integers `nums` and an integer `target`, return the indices of the two numbers such that they add up to `target`. You may return the answer in any order.
