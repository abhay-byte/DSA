Generate Sub-Strings I

Generate Sub-Strings I
Given a string, design an algorithm to generate all of its sub-strings.
We define a sub-string of an string as a contiguous part of the given string.
Example
Input : "abcde"
Output : "a", "ab", "abc", "abcd", "abcde"
"b", "bc", "bcd", "bcde"
"c", "cd", "cde"
"d", "de"
"e"

Check Good Strings

Input : "uoiea"
Output : True

Rudeness of a String

Rudeness of a String
Given a string s containing n lowercase English letters, design an algorithm to find its
rudeness _ we define rudeness of a string as the no. of pairs of integers (i, j) in s such
that 0 <= i < j <= n-1 and si = 'a' and sj = 'b'.
Example
Input : ababbaab
Output : 9

ababbaab
