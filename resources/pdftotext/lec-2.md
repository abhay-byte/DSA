Introduction to Pseudo Code

Pseudo-Code
It is a way of expressing an algorithm that involves usage of plain language to describe
the steps of an algorithm.
○ Start / Terminate
○ Variable Assignment
○ Input / Output
○ Conditional Logic
○ Loop

Check Prime
Given a positive number N ( >1 ), design an algorithm to check if the given
number is a prime number or not and express it by means of pseudo code.
Example
Input : N = 5
Output : Prime
Input : N = 4
Output : Not Prime
A number N is a prime no. if it is divisible by only 1 & the number itself.
