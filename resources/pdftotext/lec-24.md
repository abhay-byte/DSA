Replace Pi
Given a string, represented as a character array, design a recursive algorithm to
replace all the pi's in the given string with 3.14 & print the resulting string.
Example
Input : "pip"
Output : "3.14p"
Input : "pippi"
Output : "3.14p3.14"

Generate Subsequences

Generate Subsequences
Given a string, design a recursive algorithm to generate all of its subsequences.
A subsequence of a sequence is a sequence which can be obtained by removing
or more elements from the given sequence such that the relative order b/w the
sequence elements doesn't change.
Example
Input : "abc"
Output : "abc", "ab", "ac", "a", "bc", "b", "c", ""

[HW] Coin Toss

Coin Toss
Given N coins, such that each coin has two sides H (heads) and T (tails), design a
recursive algorithm to generate all the possible outcomes that we can get upon
tossing the N coins simultaneously.
Example

Input : N = 2
Output : "HH", "HT", "TH", "TT"
Input : N = 3
Output : "HHH", "HHT", "HTH", "HTT", "THH", "THT", "TTH", "TTT"

.

Restricted Coin Toss
Given N coins, such that each coin has two sides H (heads) & T (tails), design
a recursive algorithm to generate all the possible outcomes that we can get
upon tossing the N coins simultaneously such that an outcome should not
contain consecutive heads.

Example
Input : N = 2
Output : "HT", "TH", "TT"
Input : N = 3
Output : "HTH", "HTT", "THT", "TTH", "TTT"

.
