Minimum Window Substring
Given two strings S and T of lengths M and N respectively, design an
algorithm to find the length of the minimum window substring of S that
contains every character of T (including duplicates).

Example
Input : S = "adobecodebanc" T = "abc"
Output : 4
Input : S = "dboaecbbdaeba" T = "aabbbc"
Output : 8
